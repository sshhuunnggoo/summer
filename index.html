HTML,CSSの説明

HTMLはHyper Text Markup Languageで、サイトを作るために必須なもの。
CSSはCascading Style Sheetsで、簡単に言うとHTMLで書いたものを装飾するもの。

HTMLの書き方

<>で囲んだものを「タグ」と言う。<>の中にはタグ名を書き、終わりはで終わる

CSSの書き方

CSSはタグ名 {

}

サイトを作るときに必要な基本のタグ

タグ 効果
h1~6 見出し
font 文字
b 太字
i 斜め文字
br 改行
p 段落
a リンク
table 表
tr 表の行
th 表の見出し
td 表のデータ
img 画像
form フォーム
input 入力欄や送信ボタン
